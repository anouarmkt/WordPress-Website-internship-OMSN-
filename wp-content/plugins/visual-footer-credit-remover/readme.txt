== Description ==

Remove or replace the footer credit of any WordPress theme with the click of a mouse.

== Installation ==

1. From WordPress Admin, go to Plugins -> Add New
2. Search for plugin "Visual Footer Credit Remover"
3. Install and Activate
4. Go to Tools -> Visual Footer Credit Remover
5. Find and click your theme's footer credit
6. Enter content to replace footer credit
7. Click Save
